# 予想収益シミュレーション (Revenue Simulator)

## プロジェクト概要
このプロジェクトは、広告宣伝費の投入額に応じて、売上および利益を予測・可視化するシミュレーションアプリケーションです。対話的なUIを通じて、最適な広告予算の検討を支援します。

## 作成背景
本プロジェクトは、授業内で講師が作成したテキスト教材を参考に、Pythonを用いたデータ分析とWebアプリケーション（Streamlit）の実装学習の一環として作成されました。教材で示された収益計算ロジックやAltairによる可視化手法をベースに、ユーザーがパラメータを操作できるシミュレーターとして構築しています。

## 使用技術
- **言語**: Python 3.9.6
- **主なライブラリ**:
  - `Streamlit`: WebアプリケーションのUI構築
  - `pandas`: データの整理・加工
  - `Altair`: インタラクティブなグラフの作成
  - `math`: 対数を用いた収益計算

## プログラムの機能
1. **動的なパラメータ設定**: サイドバーのスライダーを使用して、広告宣伝費（1,000万円〜9,000万円）をリアルタイムに変更可能です。
2. **主要指標の算出**:
   - 入力値に基づく「予想売上」および「予想利益」の表示。
   - 利益を最大化する「最適な広告投下費用」の自動算出。
   - 全範囲における「予想最大利益」の表示。
3. **グラフ可視化**: 広告費に応じた売上と利益の推移をラインチャートで表示します。

## ファイル構成
